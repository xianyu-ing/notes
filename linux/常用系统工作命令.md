# 常用系统工作命令

date 时间：
     
     格式：date [选项] [+指定格式]
     参数：      -s  设置时间  例： 设置时间为 2019年2月28号16时32分 date -s "20190228 16:32:00"
     
     date "+%Y-%m-%d %H:%M:%S"  显示当前日期 注意 ‘+’ 号
          "+%M"   分钟
          "+%S"   秒
          "+%j"   今天中的第几
    
    
==ps 进程状态（一次性的）==
************
    ps -aux 
       -a   显示所有进程
       -u   用户信息
       -x   显示没有控制终端的进程
       
    USER：进程所有者  
    PID： 进程ID 
    %CPU: 占用CPU
    %MEM: 占用内存
    STAT: 进程状态  
            R (运行) 
            S (休眠):    触发后会脱离休眠状态
            D (不可中断)  
            Z (僵死)：  进程已经终止，进程描述还在（就是名存实亡，占个位置）
            T (停止):   进程接收到停止命令而停止运行
    COMMAND： 命令名称
    
==top 任务管理器==

![](https://snag.gy/Sv13lc.jpg)
        
    top
        users: 登录终端数
        load average: 系统负载（三个数值：1、5、15 分钟内平均值，数值越小表示负载越低）
        
    Tasks
        total: 进程总数
        running: 运行中的进程数
        sleeping: 睡眠中的进程数
        stopped: 停止进程数
        zombie: 僵死的进程数
    
    %Cpu(s)
        us: 用户空间占用 CPU 百分比
        sy: 内核空间占用 CPU 百分比
        id: 空闲 CPU 百分比
        
    KiB Mem
        total: 物理内存总量
        free:  内存空闲量
        used:  内存使用闲量
        
==kill/killall 终止指定PID进程==

    命令格式： pidof [服务名称]  查找该服务的 PID 值      
    命令格式： kill [进程PID]  
    命令格式： Killall [服务名称]  使用于一个程序多个进程的服务 如:httpd、nginx

# 工作目录切换命令
    
pwd 显示当前目录路径
   
    pwd
    
cd 跳转到指定目录
        
        格式：cd [目录名称|目录路径]
        参数：  -   返回上一次目录
                ..  返回上一级目录
                ~   跳转到家目录
                
ls|ll 显示目录文件信息
    
    格式：ls [选项][文件路径]
    格式：ll [选项][文件路径]
    
    通用参数： -a 可以显示隐藏文件
               -l 查看文件属性、文件大小
               
cat 适用查看文件内容较少的文件 
        
    格式：cat [选项][文件]
    
more 适用查看文件内容较多的文件

    格式：more [选项]文件
    操作：空格键|回车键翻页

==tail 用于查看文档最后 N 行或者持续刷新内容==
     
    格式：tail [选项][文件]
    参数：     -n  查看最后第 N 行内容  例：查看 test.log 文件最后10行内容   # tail -n10 test.log    
               -f  实时查看文件内容
               
    例：实时查看 nginx 的 access.log 最后10行记录  # tail -fn10 access.log           
               
# ==文件目录管理==

touch 创建文件或设置文件时间
    
       格式：touch [选项][文件]
       参数:       -a 修改 '读取时间'
                   -m 修改 '修改时间'
                   -d 同时修改 '读取和修改时间'  例：将 test.txt 的读取时间和修改时间 # touch -d "2019-02-28 18:04" test.txt  
                   
mkdir 创建目录

    格式：mkdir [选项] 目录名
    
cp  复制文件或目录
    
    格式：cp [选项] 源文件 目标文件
    参数：  -p 保留原始文件属性
            -r 递归复制（用于目录）
            -i 若目标存在则询问是否覆盖

mv 剪切文件或文件重命名

    格式：mv [选项] 源文件 目标路径|目标文件名
    
rm 删除文件或目录
    
    格式：rm [选项] 文件
    参数     -f 强制删除
             -r 删除目录需要
    
file 查看文件类型
    
    格式： file 文件名
    
    
# 解压/压缩与搜索

tar 对文件解压或压缩

    格式：tar [选项][文件]
    参数      -czvf  压缩
              -xzvf  解压
    
    例：压缩 www 目录  tar -zcvf  www.tar.gz  wwww
        

grep 在文本搜索关键词

    格式：grep [选项][文件]
    参数：       -n  显示行号
                 -v  反向搜索――列出没有 “关键词” 的行
                 
                 
find 用指定条件来查找文件
    
    格式：find [查找路径] 查找条件 操作
    参数：     -name  匹配名称  例：在 /etc 查找文件名以 host 开头的文件  find /etc -name "host*" -print 
               -prem  匹配权限
               -user  匹配所有者
               -group 匹配所有组
               -mtime -n +n 匹配修改内容时间 （-n 指 n 天以内，+n 指 n 天之前）
               -atime -n +n 匹配访问内容时间  （-n 指 n 天以内，+n 指 n 天之前）
               -ctime -n +n 匹配修改文件权限时间 （-n 指 n 天以内，+n 指 n 天之前）
